# Voice Diary

A self-hosted Progressive Web App for recording voice notes. Phase 1: backend MVP + PWA frontend.

## Quick Start

1. Copy `.env.example` to `.env` and set real values
2. Generate TLS certs with mkcert (see `docs/WORKFLOW.md`)
3. `docker compose up --build`
4. Open `https://localhost:8080`
